## Quick Navigation

### For Students & Regular Users
1. Start with [Catalog Guide](./catalog-guide) to find books
2. Learn [Loans](./loans-guide) and [Reservations](./reservations-guide) for borrowing
3. Set up [Profile](./profile-guide) and [Settings](./settings-guide)
4. Use [Notes](./notes-guide) and [Bookmarks](./bookmarks-guide) for personal study
5. Explore [Badges](./badges-guide) to earn achievements

### For Librarians
1. Review [Loans Guide](./loans-guide) for circulation management
2. Check [Action Logs Guide](./action-logs-guide) for activity monitoring
3. See [Orders Guide](./orders-guide) for acquisition requests
4. Manage [Digital Resources](./digital-resources-guide) and [Resource Types](./digital-resources-guide#resource-types)

### For Admins & Superusers
1. Start with [Superuser Guide](./superuser-guide) for system administration
2. Configure [Institution Settings](./institution-settings-guide)
3. Customize the [Dashboard Layout](../features/dashboard-config-guide) for each role
4. Monitor [System Logs](./system-logs-user-guide) and [Action Logs](./action-logs-guide)
5. Review [Issue Reports](./issue-reports-guide) for community feedback
6. Review [Security Features](../security/index) for best practices
